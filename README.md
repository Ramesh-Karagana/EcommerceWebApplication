# E-Commerce Web App

This is the web application for e-commerce that will have a super user side, product seller side and a user side. The super user will be able to confirm the product seller when they registered in the system and also be able to design the user and product seller sides' pages in terms of carousel image or so on . Furthermore, the super user will decide the category and subcategories that product seller side will sell their products in that categories. 
Now, the capability of the application is the CRUD operation of product, product category, product subcategory, subcategory feature and product feature.
## Technologies that I have been using;
-	Spring Boot
-   Spring Security
-	Hibernate
-	PostgreSQL
-	Restful API
-   Swagger OpenAPI
-   HATEOAS
